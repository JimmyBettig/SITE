Description du projet

Le projet consistait à développer un site web interactif permettant de gérer et d'afficher des données issues d'une base de données. Ce site devait offrir une interface intuitive pour les utilisateurs finaux tout en intégrant un back-office pour les administrateurs et développeurs. L’objectif était de structurer une base de données efficace et de concevoir une plateforme web capable de récupérer et traiter ces informations dynamiquement.

Objectifs

  • Concevoir une base de données relationnelle et l’intégrer dans un site web.
  • Développer une interface utilisateur permettant la recherche et l'affichage des données stockées.
  • Mettre en place un back-office facilitant la gestion et la mise à jour des informations.
  • Optimiser le transfert des fichiers et l’hébergement du site pour garantir un bon fonctionnement.

Technologies utilisées

PHP PHPMyAdmin HTML SFTP JavaScript

Méthodologie

Le projet s'est déroulé en plusieurs étapes : conception de la base de données via un Modèle Conceptuel des Données (MCD), création des tables avec PhpMyAdmin, puis développement du site web en PHP. Un serveur distant a été utilisé pour héberger le site, et différents outils de transfert de fichiers ont été testés pour optimiser la mise en ligne. L’approche adoptée séparait clairement le front-office (interface utilisateur) du back-office (gestion des données et développement), garantissant une meilleure modularité et maintenance du projet.

Résultats

La recherche d'un étudiant nous retourne son groupe, sa ville et la météo de cette dernière.
